[ Software for PC-9821 ]



DWP98
DARU/WAVE-PLAYER! for PC-9821 の略。

WAV / MP3 / OggVorbis / FLAC / P.M.D. / MXDRV を再生します。

PCM音源 (MATE-X、86音源、118音源および SB16/98) に対応。

ファイラーから直接再生、もしくはプレイリストに登録して再生。
リアルタイム処理のボーカルキャンセル機能もあり。

実行画面はこんな感じ。
対応環境 MS-DOS 6.2以降、Pentium 以降を搭載した PC-9821。
対応サウンドボード ・WSS-PCM (MATE-X/R,CanBe Sound,PC-9801-118)
・SB-PCM (SoundBLASTER16/98)
・86PCM (PC-9801-86,WaveStar,WaveSMIT,WaveMaster)
ソースコード公開 なし
ダウンロード 2012/09/06　vector




OPL3SA3 Control
OPL3SA3 こと YMF715 のエフェクト処理を MS-DOS で利用します。

マスターボリューム、スピーカーモード、3Dエフェクト(WIDE,BASS,TREBLE)
が設定可能です。

なお、独自解析による産物なので実行した結果、何が起きても責任は取れません。
対応環境 MS-DOS 6.2以降、PC-9821。
対応サウンドボード YMF715 を搭載した
・PC-9821V166,V200,V233 の後期型(青札)
・PC-9821Nr150 等の後期型
(確認はしていませんが YMF701 では動作しません。)
ソースコード公開 あり（Turbo C++ or OpenWatcom）
ダウンロード 2011/09/16　opl3sa3.lzh




ISA PnP Tolls 1.27 for PC-9821 (α版)
Ｃバスの PnP ボードの設定を試みるソフトです。
Windows でしか使えないボードが DOS でも使えるようになるかも知れません。

AT互換機用ソフトの ISA PnP Tools を PC-9821 に移植しました。
(名前を変えようかと思いましたがいろいろ面倒なので止めました。)

人柱ソフトウェアです！
(使用は全て自己責任でお願いします。)

※IF-SEGA/98 の初期化が可能。
※118音源(PnPモード) の初期化が可能。
※SB AWE32/98 の初期化が可能。
対応環境 MS-DOS と Plug and Play に対応している PC-9821。
ソースコード公開 あり（OpenWatcom）
ダウンロード 2011/11/14　isapnp98.lzh




SW2ATARI
SideWinder GamePad (初代) を VEM486 の I/O Trap 機能を使って
PC-9801-26K,PC-9801-86 に接続された ATARI 仕様の
２ボタンジョイパッドとして使用できます。
対応環境 MS-DOS 6.2以降
対応メモリマネージャ VEM486
ソースコード公開 あり（Turbo Assembler）
ダウンロード 2012/09/29　sw2atari.lzh
