2017/06/04 - Intel のグラフィックドライバが更新できない

Windows 7(64 bit) でノートPCの CPU 内蔵のグラフィック機能を
使うためにインテルからドライバをダウンロードして
インストールしようとしたら CPU も OS も間違っていないのに

「システム要件を満たしていません」

のエラーが出てインストールできない…

とりあえずメーカーのドライバだとインストールできるが古い。

いろいろ試して無理やりインストールできたのでその方法。

1.インテルから zip 版のドライバをダウンロードして解凍しておく。
2.メーカードライバをアンインストールする。
　→ VGA標準ドライバが入る。
3.デバイスマネージャのVGA標準ドライバのプロパティを開いて、
　「ドライバの更新」から解凍したインテルドライバの
　「graphics」フォルダを指定する。
4.ドライバがインストールされる。

setup.exe の判定に問題があるんじゃないかなぁ…


2017/05/20 - カーオーディオの AAC 対応

クルマのＵＳＢメモリ音楽再生環境ってどうなってるのかなと検索。
CD/DVD には対応しないUSBメモリ、ラジオのみモデルだとで１万円以下で買えるのを発見。

パイオニア、スマホから選曲や音質設定できる1DINカーオーディオ - AV Watch
http://av.watch.impress.co.jp/docs/news/1022612.html

全機種 AAC 対応って書いてあるんだけど…
MVH-3300 取扱説明書 発売順現在～ AVメインユニット カーAV パイオニア株式会社
http://pioneer.jp/support/s-manual.php?product_no=00009213&disp=s-year.php&sort=1

製造元のマニュアルを見ると MVH-3300 は AAC に対応していない。
上位機種の MVH-5300 なら Bluetooth と AAC に対応していて約1.5万円くらい。
他のメーカーも Bluetooth 対応モデルは AAC に対応しているっぽい。

各製品の仕様を見ると AAC と書いてあっても m4a 形式にしか
対応していないようなので注意が必要。


2017/01/01 - Matrox G550

AGP/PCI 接続の古いグラフィックチップ。
製品としてはブリッヂを搭載して PCIe x1 版も存在する。

DirectX 6 世代のチップで今では使えるレベルではない。

一応、Windows7 のドライバもリリースされているが、
Vista 以降は 32bit ですら DirectX(HAL)、OpenGL(ICD) もサポートしていない。
アクセラレーションが効くのは XP まで…？

解像度の選択が少ないので調べてみると、

出力先 解像度
アナログ(プライマリ) 2048 x 1536
アナログ(セカンダリ) 1600 x 1200
デジタル(DVI) 1280 x 1024
TV 2048 x 1536

デジタル出力が HD(1920x1080) に対応していないゾ…。
マルチディスプレイ用の追加グラフィックカードにも使えない。
