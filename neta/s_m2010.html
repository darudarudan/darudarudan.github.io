2010/12/21 - ASUS Xonar DS

搭載チップは ASUS AV200 で PCI 接続、ロープロ対応。

ASUS の AV 系チップは CMI のチップと仕様が似通っているので
CMI 系のカスタムチップなのかも知れない。
(下位モデルの ASUS Xonar DG は CMI8786 を搭載しているし。)

この製品の良さは機能に対してのコストパフォーマンス。

昔だったら数万円したオーディオカードを買わないと使えない機能を
わずか数千円で提供してくれる。
（あくまで幅が広いのであって深さがあるかは別。）

再生 192Kz/24bit、録音 96Kz/24bit、
S/P DIF 44～192KHz/24bit Dolby Digital, DTS, WMA Pro
に対応している。
付属品でピンジャックto角型アダプタも付いてるのでコンポ接続も簡単。

さらにいろいろな API に対応しているので試すのは楽しいかも。

GX 2.5

2010/10/26 - Intel 統合グラフィクで OpenGL 2.0

Q965 は DirectX 9.0c に対応しているんだから OpenGL 2.0 に
対応してても良さそうだけど OpenGL 1.4 止まり。

US15W(GMA500)は組み込み向けのIEGDドライバを入れると
OpenGL 2.0 に対応するというウワサがあるので
Q965 で試してみたら Microsoft 「GDI Generic」で撃沈。

海外だと 965 の Driver 7.15.10.1666 で OpenGL 2.0 とある。
インテルのサイトで該当ドライバを探すと…Vista用…。

つまりXP用ドライバだと最新にしても OpenGL 1.4 が限界と…

2010/09/18 - OpenGL ES 2.0 対応ドライバ

●GeForce 8200 → 260.63_desktop_winxp_32bit_international_beta に更新。
GL_ARB_ES2_compatibility
GL_ARB_get_program_binary
WGL_EXT_create_context_es2_profile
シェーダーバイナリとコンテキストが対応になってる。

拡張はいくつか増えたけど OpenGL 2.1.2 のまま。GLES2 のサポートは無し。
ハードウェア的には ES2 をサポートしても良いと思うけど…。

サポートしないということかな？

●ATI は CATALYST 10.8 以降
ES2 関連に対応したらしいけど使用可能な API 判断は開発者任せ？
良い点は atioglxx.dll から EGL API を取り出して使用可能なので
EGL を使うならモバイル環境とソース互換にしやすいかも。

2010/05/29 - Athlon II X2 260u @ 25W

新たなＣＰＵをゲットしたのでワットチェッカーで消費電力を測定してみた。

対象ＣＰＵはコレ。
ＣＰＵ
Phenom 9350e 2.0GHz 4core L2 - 512KB×４ L3 - 2048KB (共有) 65W
Athlon II X2 260u 1.8GHz 2core L2 - 1024KB×２ L3 - 0KB （なし） 25W

環境はこんな感じ。
ハードウェア仕様
M/B GA-MA78GPM-DS2H
Memory DDR2-1066 2GB 1.8v ×2
HDD Hitachi HTS545025B9A300
DVD Optiarc AD-7170A
Power VariusEX 350W

そんでもって早速テスト。
測定
ＣＰＵ Phenom 9350e Athlon II X2 260u
起動 84W 58W
アイドル 60W 50W
Superπ×２ 90W @ 17sec 62W @ 18sec
Superπ×４ 112W @ 17sec 62W @ 35sec
※ 動的クロック制御はONにしているのでクロックが最低の状態から測定。
規模の小さい Superπ だとほとんど差が出ないのにビックリ。
クロックは 200MHz も低いのに 260u と差が少ないのは
L2 のアクセス速度が関係しているのかもね。

でも消費電力を単純計算すると Phenom 9350e って 65W オーバーしてるような…。

2010/04/18 - AMD 系で LAN が２つある Mini-ITX マザー

Jetway NC81-LF
65W までの AM2+ CPU に対応。AMD780G+SB700 で LAN×２、DVI/HDMI。
残念なのはシリアルポートがI/Oシールドにないこと。
(内部から付属ケーブルで外には出せる)
現時点での最新 BIOS は 2010-3-19 A10

65W までの AM2+ CPU に対応。MCP78S で LAN×２、シリアルポート×２、DVI/HDMI。
残念なのはチップセットがファン付きなこと。
また、DIMMスロットが１本なのでデュアルチャンネル動作にならない。
現時点での最新 BIOS は 2009-11-27 A09
